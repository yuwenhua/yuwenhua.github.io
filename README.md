## 二、服务内容与限制

### 2.1 服务范围

我们向用户提供以下设备管理相关服务：

* 设备注册与管理：支持通过Apple官方流程（如ABM/ASM）将苹果设备纳入MDM管理体系，实现设备激活、配置文件下发、远程擦除/锁定等功能；
* 策略配置：允许设备所有者通过管理后台设置设备使用规则（如应用安装限制、网络访问控制、密码策略、定位开关等）；
* 数据统计：提供设备状态（在线/离线）、合规性（是否违反策略）、应用使用情况等基础监控信息；
* 技术支持：针对设备管理功能使用问题提供有限的远程协助（不含设备硬件故障或苹果系统本身的技术问题）。

### 2.2 服务限制

* 本软件仅适用于合法拥有苹果设备的主体（如企业、学校等），禁止用于非法获取他人设备控制权、监控个人隐私等违规用途；
* 我们不直接控制或操作用户设备，所有管理行为需由设备所有者通过管理后台主动发起；
